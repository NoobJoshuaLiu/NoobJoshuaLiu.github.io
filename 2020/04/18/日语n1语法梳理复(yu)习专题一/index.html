日语n1语法梳理复(yu)习专题一

发表于 2020-04-18 | 分类于 日本語

掐指一算，距离N1考试只有两个月出头了QAQ，然鹅自己里很是没底

词汇听力可以天天练，且不说效果怎样，至少得着一个心理上的安慰。语法就不同了，众所周知，日语语法到了N2以后，句型日益复杂，大量高度相似的句型的出现使得语法难度进一步上升。很多时候靠字面意思猜也猜不出个所以然，去猜反而容易翻了车，猜出的意思和本意差了十万八千里。

死记硬背已然不能奏效，多看多练才是王道！TAT

DAY 1

  • XがXなら、YもYだ

    X = Noun 1, Y = Noun 2
    Meaning: Used to say two things are similar (but only in a negative way).
    It is very similar to the English expression like [father] like [son], though this grammar is point is only used to express something negative that is similar.
    Two nouns are similar in some negative regard.

    例文：
    ア：子供が子供なら、親も親だ。

    意味（用鼠标选定本行文字以查看答案）：子供も親も両方良くない

    イ：あのカフェは、味も味なら、サービスもサービスだ。

    意味（用鼠标选定本行文字以查看答案）：味もサービスも両方良くない

    汉语总结：X和Y都很差
    直译：提到X（差）的话，Y也一样（差）。

  • 相似语法1：だの～だの

    meaning：and; and the like; and so forth
    This grammar point is used to list multiple examples, usually to express dissatisfaction.
    大辞林：体言または活用語の言い切りの形に付いて、物事を並列・列挙する。
    一般に「…だの…だの」の形で用いられる。
    ① 話し言葉として使われる。
    ② 話し手の不満や非難の気持ちが含まれる。

    例文：
    ア：僕の毎月の小遣いは、飲み会だのカラオケだので消えていく。
    意味（涂黑查看）：僕の毎月の小遣いは、飲み会やカラオケや色々なもので消えていく

    イ：彼女は、風邪を引いただの、頭が痛いだのと言って、よく授業を休む。
    意味（涂黑查看）：風邪を引いたとか、頭が痛いとか、ほかにもいろいろ

    汉语总结：
    不完全列举，列举两个代表性的事物来指代全体；
    一般用于口语场合，说话人通常用该句型表达抱怨等负面情绪。 　

  • 相似语法2：Xといい～Yといい

  • 相似语法3：Xやら～Yやら

  • 相似语法4：Xなり～Yなり

  • 相似语法5： X+ といわず + Y + といわず

DAY 2

背诵以下语法及例句

  • ～や・～や否や

    ．．．わたしはその人の顔を一目見るや、30年前に別れた恋人を気が付いた。
    ．．．選挙戦が始まるや否や、あちこちからにぎやかな声が聞こえてきた。

  • ～が早いか

    ．．．海外旅行に出発の日、山田さんは空港に着くが早いか、コンビニに駆け込んだ。
    ．．．うちの子はいつも学校から帰ってきて、カバンをほうりだすが早いか、遊びにいってしまう。

  • ～なり

    ．．．彼はコーヒーを一口飲むなり、吐き出してしまった。（未预料到的、不好的结果）

  • ～そばから

    ．．．毎日返事を書くそばから次々に新しいメールが来る。
    ．．．このデキストは漢字が多くて大変だ。調べたそばから新しい漢字が出てくる。

    （反复的，消极的结果）

  • ～にあった

    ．．．駱駝はは乾燥地にあって、こぶに栄養を蓄えることによって生き延びているである。
    ．．．明治時代の初め、日本は正に発展途上期にあって、みな生き生きとしていた。

  • ～てからというもの（は）

    ．．．娘が大学に入り家を出て行ってからというもの、家の中が寂しくなった。
    ．．．子供が生まれてからというもの、子供のおもちゃばかり見て歩いています。

    （句型后跟变化，该变化到说话时一直持续）
